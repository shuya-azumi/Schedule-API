# 面接 日程調整サービス（MVP）

人材紹介の3者（**求職者・エージェント・事業所**）が、チャット形式で面接日程を調整するための最小プロトタイプ。
エージェントが候補日時を打診し、求職者・事業所が「参加可／不可」を選ぶ。全員が参加可になった枠が緑にハイライトされ、エージェントが確定する。やり取りはリアルタイムで全員に同期される。

## 技術
- HTML / CSS / JavaScript（jQuery）
- Firebase **Realtime Database**（v9 モジュール・CDN読み込み）
- ビルド工程なし

## 主な機能
- 役割（求職者・エージェント・事業所）を選んで参加。URLで同じルームを共有
- エージェントが候補日時を追加（日付ピッカー＋15分刻みの時刻、時間順に自動整列）
- 求職者・事業所が「参加可／不可」を回答 → 全員参加可で緑ハイライト
- エージェントが確定（別の枠への切替・確定解除も可能）
- リアルタイムのチャット ＋ 定型文ワンクリック送信（宛名の自動差し込み付き）

## 使い方
1. ローカルで動かす場合は **HTTPサーバー経由**で開く（VS Code の Live Server、または `python -m http.server`）。
   ※JavaScript をモジュール（`js/app.js`）として読み込むため、`index.html` のダブルクリック（`file://`）では動きません。
2. 名前と立場を選んで参加
3. 右上「招待URLをコピー」で URL（`?room=...`）を3者に共有 → 同じルームに入れる
4. エージェントが候補を追加 → 各自が参加可／不可 → 全員参加可で確定

## セットアップ（自分のFirebaseで動かす場合）
1. Firebase で Realtime Database を作成
2. `js/app.js` の `firebaseConfig` を自分のプロジェクトの値に差し替え（`databaseURL` 必須）
3. 検証中はデータベースのルールをテストモードにする
